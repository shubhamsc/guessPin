rules and guidelines:

  1. You are allowed 6 attempts to enter a four-digit PIN code.
    After 6 attempts, the PIN code resets to a random (other) one.
  2. Pin doesn't have repeated digits(e.g. 9981,5555 etc).
  3. If the correct digit is in the correct place, that digit will be green.
  4. If the correct digit (i.e. a digit that is in the actual PIN) is in the incorrect place, that digit will be yellow.
  5. If an incorrect digit is entered (i.e. a digit that is not in the actual PIN), that digit will be red.
